Class DataSelection

A DataSelection is a set of data that the user is working with. It is copied into the underlying Arrow files and available to the GPU, so it should not be abused; as a rule of thumb, it's OK to create these in response to user interactions but it shouldn't be done more than once a second or so.

applyToAllLoadedTiles

  • applyToAllLoadedTiles(): Promise<void>

  • Ensures that the selection has been evaluated on all tiles loaded in the deeptable. This is useful if, for example, your selection represents a search, and you are zoomed in on one portion of the map; this will immediately execute the search (subject to delays to avoid blocking the main thread) on all tiles that have been fetched even if out of the viewport.

    Resolves upon completion.
